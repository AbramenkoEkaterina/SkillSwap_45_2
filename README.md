# SkillSwap_45_2

## 📖 Описание проекта: 
SkillSwap — это платформа обмена навыками «Я начинаю», где пользователи могут делиться своими умениями и находить людей для обучения и сотрудничества.
Проект реализован в командной разработке.

**👥 Команда:**
13 человек (включая 1 backend-разработчика)

## 🚀 Установка и запуск
**1️⃣ Клонирование репозитория**
```
git clone git@github.com:AbramenkoEkaterina/SkillSwap_45_2.git
```
2️⃣ Переход в папку проекта
```
cd SkillSwap_45_2
git checkout develop
```
3️⃣ Установка зависимостей
```
npm install
```
4️⃣ Запуск проекта
```
npm run dev
```

Приложение будет доступно по адресу:

http://localhost:5173
